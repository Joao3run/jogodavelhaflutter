# Jogo da Velha

# Sobre o projeto


 Jogo da velha é uma aplicação construída com o objetivo de praticar o desenvolvimento na SDK Flutter, uma ferramenta de desenvolvimento híbrido para Android e iOS, feita pelo Google.
